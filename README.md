# 🎓 EduLens AI

**EduLens AI** is an AI-powered student performance analytics platform that helps educators and institutions identify at-risk students, understand academic trends, and generate actionable recommendations using Machine Learning and AI reasoning.

## 🚀 Live Demo

🌐 https://edulens-ai.streamlit.app

## 📊 What EduLens AI Can Do

* Predict students who may be academically at risk
* Analyze student performance data
* Generate AI-powered insights and recommendations
* Visualize trends through interactive dashboards
* Provide step-by-step reasoning for decisions
* Export AI-generated reports

## 🛠️ Technology Stack

* Python
* Streamlit
* Pandas
* Scikit-learn
* Plotly
* GitHub Models
* Microsoft Azure

## ▶️ How to Use

### Step 1

Open the live application.

### Step 2

Download and upload the sample dataset.

### Step 3

Explore student analytics, risk predictions, and visualizations.

### Step 4

Ask questions such as:

* Which students are most at risk?
* What factors affect performance the most?
* How can attendance be improved?
* What interventions should be recommended?

### Step 5

Download the AI-generated report.

## 🧠 AI Reasoning Workflow

EduLens AI follows a structured reasoning process:

### Step 1 – Analyze

Examines student performance data and identifies important patterns.

### Step 2 – Finding

Highlights key insights, trends, and risk indicators.

### Step 3 – Recommendation

Provides actionable recommendations to improve student outcomes.

## 📈 Dataset

This project uses the **UCI Student Performance Dataset** for demonstration and analysis.
